==========
User Guide
==========


(soon™ incomplete guide)


Multiplied focuses on combinational multiplication, fine-grained control over algorithms, and the choice of making algorithms entirely by hand or using build-in building blocks.

Recommended resources before starting:


.. hlist::
    :columns: 4

    * `Binary Multipliers <https://en.wikipedia.org/wiki/Binary_multiplier>`_
    * `Wallace Tree <https://en.wikipedia.org/wiki/Wallace_tree>`_
    * `Dadda Tree <https://en.wikipedia.org/wiki/Dadda_multiplier>`_
    * `Teman, A. Fast Multipliers <https://youtu.be/4FwESTOVT-o/>`_
